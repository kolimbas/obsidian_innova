## 🌐 Desarrollo Web & E-commerce

**Lovable MCP Server** — Conecta Claude Code con Lovable. Crear proyectos, iterar y deployar sin salir del agente. Instalar en Claude Code → Settings → Connectors → `https://mcp.lovable.dev`

**Lovable Themes** — Definir paleta y tipografía del cliente una vez, reutilizar en todas las iteraciones. Crear el Theme antes del primer prompt para evitar que Lovable invente colores genéricos.
`https://lovable.dev/blog/mcp-servers`

**Supabase Stripe Sync Engine** — One-click en el dashboard de Supabase. Sincroniza clientes, suscripciones e invoices de Stripe a tablas Postgres. Queries SQL directas en vez de llamadas a la API de Stripe.
`https://supabase.com/blog/stripe-sync-engine-integration`

**Lovable Test/Live Environments** — Dos entornos con DB aisladas. Siempre trabajar en Test antes de tocar Live. Evita romper datos del cliente en producción.
`https://lovable.dev/blog/lovable-2-0`
